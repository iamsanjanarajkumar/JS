JS-DATATYPES

        DATATYPE
        1.Primitive data type
        2.Non-primitive data type
      1.PRIMITIVE DATA type
     -Int(integer) ->It accepts only round numbers ex= 0-9
     -Float(float) ->It accepts decimal values ex= 10.6
     -Boolean(boolean) ->It accepts either true or false 
     -char(char) ->It accepts single character ex= apple
     -void -> returns null values
     
     2.NON-PRIMITIVE/DERIVED  DATA type
     - Array
     -Structure
     -Union
     -Enum
     -Pointer
     -Function
